# Machine Learning - AdventureWorks BI Project

This folder contains machine learning scripts to enhance the BI project!

---

## ML Scripts:

### 1. Customer Segmentation (`customer_segmentation.py`)
Uses K-Means clustering to segment customers into groups based on:
- Age
- Total Orders
- Total Quantity Purchased

**How to run:**
```bash
python ML/customer_segmentation.py
```

---

### 2. Sales Forecasting (`sales_forecasting.py`)
Uses Linear Regression (with lag features) to forecast future monthly sales!

**How to run:**
```bash
python ML/sales_forecasting.py
```

---

## Results:
All plots and visualizations are saved in the `ML/results/` folder!
